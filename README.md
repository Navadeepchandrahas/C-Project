# Currency Converter Project

## Objective
This project is a simple C program that converts currency values based on predefined exchange rates.

## How to Compile
gcc currency_converter.c -o currency_converter

## How to Run
./currency_converter
(Windows: currency_converter.exe)

See the screenshot uploaded in this repository.

## Notes / Assumptions
- The program uses fixed conversion rates.
